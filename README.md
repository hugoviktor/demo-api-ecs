# node-express-mongo-app
Rest API demo using node,express and mongoDB by IT Jugadu.

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Run Server

```bash
node server.js
```

You can browse the apis at <http://localhost:3000>
